# AutoSpec AI Frontend

Next.js frontend for AutoSpec AI.

For full project setup, backend instructions, environment variables, and deployment steps, see the root README:

[../../README.md](../../README.md)

## Local Run

```bash
npm ci
cp .env.example .env.local
npm run dev
```

Required local env:

```bash
NEXT_PUBLIC_API_URL=http://localhost:8000
```
